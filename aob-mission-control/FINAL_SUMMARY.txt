================================================================================
API ENDPOINTS
================================================================================

Authentication:
  GET/POST  /api/auth/[...nextauth]      OAuth flow & callbacks

Gmail Integration:
  GET       /api/drafts                    List all Gmail drafts
  GET       /api/drafts/[id]               Get draft full content
  POST      /api/drafts/[id]               Send draft via Gmail

Google Sheets:
  GET       /api/pipeline                  Fetch B2B Pipeline data

Inbox:
  GET       /api/inbox                     Get recent email items

All endpoints require authentication (checked via session).
